4. üöÄ Installation Guide

Prerequisites

  • Flutter SDK 3.8.1 or higher
  • Android Studio or VS Code
  • Firebase account
  • Git

Step-by-Step Installation

Step 1: Clone Repository

git clone <repository-url>
cd quiz_quest

Step 2: Install Dependencies

flutter pub get

Step 3: Firebase Setup

  1. Create Firebase project at console.firebase.google.com
  2. Add Android/iOS apps
  3. Download google-services.json (Android)
  4. Download GoogleService-Info.plist (iOS)
  5. Enable Email/Password authentication
  6. Create Firestore database

Step 4: Run Application

flutter run
‚úÖ Success: If everything is set up correctly, the app will launch on your connected device or emulator.
